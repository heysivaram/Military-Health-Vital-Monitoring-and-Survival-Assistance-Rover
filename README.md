# Military Health Vital Monitoring and Survival Assistance Rover

## Project Overview

This project is a defense oriented intelligent system designed to monitor the health condition of soldiers and provide survival assistance during battlefield operations.

The system integrates a wearable health monitoring band with a semi autonomous surveillance rover.

The wearable band measures vital parameters such as heart rate, blood oxygen level and body temperature.

The rover acts as the first responder in hazardous environments by performing environmental monitoring, threat detection and location tracking.

---

## Key Features

• Soldier health monitoring band  
• Heart rate and SpO2 measurement  
• Body temperature monitoring  
• Environmental hazard detection  
• Semi autonomous surveillance rover  
• GPS location tracking  
• Battlefield monitoring support
